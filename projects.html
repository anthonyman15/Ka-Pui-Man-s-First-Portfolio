Project #1

First Java UI Project - Final Year Module Selection

This is my university assignment project that I had previously completed using object-oriented programming style method. This program allows students to select their final modules by choose the enrolled course.

Both courses have cap to max 60 credits for each semester, both courses will have their default selected courses, Development Project and Systems Building: Methods for Computer Science and Development Project, Systems Buidling: Methods and Rigorous Systems for Software Engineering. Once the course type has been selected, user will be able to select their prefer modules, as modules credits capped at 120 in total, 60 for each semester, beside with default modules counted towards to the calculated credits. System prevents user select extra modules more than needed. Reserve modules allows user to select modules if the selected is fulled, in reserve modules page it filtered out the already selected modules from the previous page. Finally user can overview their selection in the overview page with their entered student details, and be able to save the overivew for next time access.
